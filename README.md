# Applications interactives - Gestion financiere L2 AES

Mini-site statique pour publier les applications HTML du cours.

## Avertissement

Ces applications sont diffusees a des fins exclusivement pedagogiques dans le cadre du cours de Gestion financiere en L2 AES.

Les donnees, exemples, visualisations et parametres presentes sont des instantanes indicatifs ou des illustrations simplifiees. Ils peuvent etre incomplets, approximatifs ou non actualises.

Ces contenus ne constituent ni un conseil financier, ni une recommandation d'investissement, ni une information de marche en temps reel.

Contenu :
- `index.html` : page d'accueil avec liens vers les applications
- `gestfin_theorie-financiere_v2.html`
- `gestfin_financement_v2.html`
- `gestfin_investissement_v2.html`
